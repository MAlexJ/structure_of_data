------------------------------------------------
Stack
------------------------------------------------

Stack is an abstract dta type that servers as a collection of elements, with two principal operation:
    - push(), which  adds a new element to the collection;
    - pop(), which extracts and removes the most recently element that wos not yet removed;

    Also, peek() operation may give access to the top element without modify the stack.

The order in which elements come off a stack gives rise to its alternative name, LIFO (last in, first out)
